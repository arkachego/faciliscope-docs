# Navigation Guide

This guide will help you understand how to navigate the FaciliScope application and find the features you need.

## Main Navigation Menu

The main navigation menu is located on the left side of the screen. It contains the following main sections:

### Dashboard
The dashboard provides an overview of your facility operations:
- **Home** - Main dashboard with key metrics and widgets
- **Meters** - Energy consumption monitoring
- **Spaces** - Space utilization analytics
- **Costs** - Cost analysis and budgeting overview
- **Assets** - Asset overview and statistics
- **Work Orders** - Work order metrics and status
- **Risks** - Predictive maintenance and risk analysis

### Work Orders
Manage all maintenance and repair tasks:
- **All Work Orders** - View and manage all work orders
- **Templates** - Create and manage work order templates

### SLA (Service Level Agreements)
Monitor and manage service level agreements:
- **Dashboard** - SLA metrics and status overview
- **Definitions** - Create and manage SLA definitions

### Budgets
Financial management and cost tracking:
- View and manage budgets
- Track transactions and spending
- Analyze budget performance

### Purchase Orders
Procurement and purchasing:
- Create and manage purchase orders
- Track order status and approvals
- Manage vendor relationships

### Assets
Asset management:
- **All Items** - View and manage all assets
- **Categories** - Organize assets by category

### Maintenance
Maintenance scheduling and execution:
- **Schedules** - Manage maintenance schedules
- **Calendar** - View maintenance calendar

### Inventory
Inventory and parts management:
- **All Items** - View and manage inventory items
- **Facilities** - Inventory by facility
- **Spaces** - Inventory by space

### Other Sections
- **Meters** - Energy meter management
- **Vendors** - Vendor and contractor management
- **Documents** - Document management and storage
- **Users** - User and role management (Admin only)

## Page Layout

Most pages in FaciliScope follow a consistent layout:

### Header
- **Breadcrumbs** - Shows your current location in the application
- **Page Title** - The name of the current page
- **Action Buttons** - Common actions like Create, Edit, Delete

### Content Area
- **Data Tables** - Lists of resources with filtering and sorting
- **Forms** - For creating and editing resources
- **Detail Views** - Comprehensive information about a specific resource
- **Tabs** - Organized sections of related information

### Filters and Search
Most list pages include:
- **Search Bar** - Search by name, description, or other fields
- **Filters** - Filter by status, priority, date range, etc.
- **Sorting** - Sort by various columns
- **Pagination** - Navigate through multiple pages of results

## Common Actions

### Creating Resources
1. Navigate to the resource list page (e.g., Work Orders, Assets)
2. Click the "Create" or "New" button (usually in the top right)
3. Fill out the form with required information
4. Click "Save" or "Create"

### Viewing Details
1. Navigate to a list page
2. Click on a resource name or use the "View" action
3. Review all details in the detail view
4. Use tabs to see related information

### Editing Resources
1. Navigate to the detail view
2. Click the "Edit" button
3. Modify the information
4. Click "Save" or "Update"

### Filtering and Searching
1. Use the search bar to find specific items
2. Apply filters using the filter options
3. Combine multiple filters for precise results
4. Clear filters to reset the view

## Keyboard Shortcuts

- **Ctrl/Cmd + K** - Quick search (if implemented)
- **Esc** - Close modals and dropdowns
- **Enter** - Submit forms

## Mobile Navigation

On mobile devices:
- The navigation menu collapses into a hamburger menu
- Tap the menu icon to expand/collapse
- Swipe gestures may be available for navigation
- Touch-optimized buttons and controls

## Tips for Efficient Navigation

1. **Use Breadcrumbs** - Click on breadcrumb items to navigate back
2. **Bookmark Common Pages** - Use browser bookmarks for frequently accessed pages
3. **Use Search** - The search function helps you find resources quickly
4. **Learn Keyboard Shortcuts** - Speed up your workflow
5. **Customize Dashboard** - Arrange widgets to show the information you need most
