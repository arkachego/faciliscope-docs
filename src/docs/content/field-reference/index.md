## Field Categories

### Required Fields
Fields that must be filled when creating a resource.

### Recommended Fields
Fields that should be filled for best results.

### Optional Fields
Fields that can be left empty.

### Calculated Fields
Fields that are automatically calculated by the system (read-only).

### System Fields
Fields automatically managed by the system (read-only).

## Field Types

### Text Fields
- **String**: Single-line text
- **Text**: Multi-line text area
- **Email**: Email address format
- **URL**: Web address format

### Number Fields
- **Integer**: Whole numbers
- **Float**: Decimal numbers
- **Currency**: Monetary values

### Date/Time Fields
- **Date**: Date only
- **DateTime**: Date and time
- **Time**: Time only

### Selection Fields
- **Enum**: Predefined options
- **Dropdown**: Select from list
- **Multi-select**: Select multiple options

### Reference Fields
- **UUID**: Reference to another resource
- **Relation**: Related resource object
