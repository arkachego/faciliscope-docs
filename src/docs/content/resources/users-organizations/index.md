# Users & Organizations

Complete guide to Users & Organizations in FaciliScope.

## Overview

Users & Organizations manage user accounts, roles, permissions, and organizational structure in FaciliScope.

## What are Users & Organizations?

Users are individuals who use the system, while Organizations represent companies or entities that use FaciliScope. Users belong to organizations and have specific roles and permissions.

## Key Features

- **User Management**: Create and manage user accounts
- **Role-based Access**: Assign roles with specific permissions
- **Organization Management**: Manage organizational structure
- **Multi-tenancy**: Support multiple organizations
- **Permission Control**: Fine-grained permission control

## Getting Started

1. Navigate to **Users & Organizations** in the main menu
2. Click **Create User** to add a new user
3. Assign roles and permissions
4. Link to organization
5. Configure user settings
