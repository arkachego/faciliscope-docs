# SLA Definitions

Complete guide to SLA Definitions in FaciliScope.

## Overview

SLA Definitions define service level agreements that specify response and resolution times for work orders. They help ensure work is completed within agreed timeframes.

## What are SLA Definitions?

SLA Definitions specify the expected response and resolution times for work orders based on priority, type, or other criteria.

## Key Features

- **Response Time Rules**: Define response time requirements
- **Resolution Time Rules**: Define resolution time requirements
- **Priority-based SLAs**: Different SLAs for different priorities
- **Automatic Application**: Automatically apply SLAs to work orders
- **Compliance Tracking**: Track SLA compliance

## Getting Started

1. Navigate to **SLA Definitions** in the main menu
2. Click **Create SLA Definition** to add a new SLA
3. Define response and resolution times
4. Set priority and type criteria
5. Enable automatic application
