# Inventory

Complete guide to Inventory in FaciliScope.

## Overview

Inventory represents parts, materials, and supplies used in maintenance work. Tracking inventory helps manage stock levels, locations, and supports work orders.

## What is Inventory?

Inventory items are parts, materials, or supplies that are used in maintenance work or stored for future use. They can be consumed or tracked as assets.

## Key Features

- **Stock Level Tracking**: Monitor current stock levels
- **Location Management**: Track where items are stored
- **Work Order Integration**: Link items to work orders
- **Reorder Points**: Set minimum stock levels
- **Cost Tracking**: Track inventory costs

## Getting Started

1. Navigate to **Inventory** in the main menu
2. Click **Create Inventory Item** to add a new item
3. Enter item details (name, SKU, category)
4. Set initial stock level
5. Define storage location
