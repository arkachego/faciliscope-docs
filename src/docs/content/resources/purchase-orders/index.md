# Purchase Orders

Complete guide to Purchase Orders in FaciliScope.

## Overview

Purchase Orders are procurement documents that track vendor orders, manage approvals, and link to budgets.

## What are Purchase Orders?

Purchase Orders document requests to purchase goods or services from vendors. They track the procurement process from request to delivery.

## Key Features

- **Vendor Management**: Link to vendor information
- **Approval Workflow**: Manage approval processes
- **Budget Integration**: Link to budgets and track spending
- **Status Tracking**: Track order status from request to delivery
- **Cost Management**: Monitor procurement costs

## Getting Started

1. Navigate to **Purchase Orders** in the main menu
2. Click **Create Purchase Order** to add a new order
3. Select the vendor
4. Add items to purchase
5. Submit for approval
