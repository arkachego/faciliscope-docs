# Common Workflows

Step-by-step guides for common workflows in FaciliScope.

## Work Order Workflows

### Creating and Completing a Work Order

1. **Create Work Order**
   - Navigate to Work Orders
   - Click "Create Work Order"
   - Fill in required fields
   - Assign users
   - Save

2. **Assign Work**
   - Open work order
   - Assign to technician(s)
   - Notify assigned users

3. **Execute Work**
   - Technician updates status to "In Progress"
   - Starts time tracking
   - Performs work
   - Adds comments and photos

4. **Complete Work**
   - Technician updates status to "Completed"
   - Stops time tracking
   - Adds completion notes
   - Attaches completion photos

5. **Close Work Order**
   - Manager reviews
   - Updates status to "Closed"
   - Asset maintenance history updated

### Preventive Maintenance Workflow

1. **Create Maintenance Schedule**
   - Navigate to Maintenance Schedules
   - Create schedule for asset
   - Set frequency (monthly, quarterly, etc.)
   - Assign users

2. **Automatic Work Order Generation**
   - System generates work order when due
   - Work order appears in list
   - Assigned users notified

3. **Execute Maintenance**
   - Technician opens work order
   - Completes checklist
   - Documents work performed
   - Updates status

4. **Schedule Next Maintenance**
   - System calculates next due date
   - Schedule updated automatically

## Asset Management Workflows

### Adding a New Asset

1. **Create Asset Record**
   - Navigate to Assets
   - Click "Create Asset"
   - Fill in asset information
   - Link to facility
   - Assign users

2. **Set Up Maintenance**
   - Create maintenance schedules
   - Link to asset
   - Set frequencies

3. **Track Asset**
   - Monitor work history
   - Update status as needed
   - Track costs

## Budget Management Workflows

### Creating and Managing a Budget

1. **Create Budget**
   - Navigate to Budgets
   - Click "Create Budget"
   - Set period and amounts
   - Allocate to categories

2. **Track Spending**
   - Work orders record costs
   - Purchase orders record costs
   - System tracks vs. budget

3. **Monitor Budget**
   - Review budget vs. actual
   - Set alerts
   - Adjust as needed
