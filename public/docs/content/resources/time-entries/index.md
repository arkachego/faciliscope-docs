# Time Entries

Complete guide to Time Entries in FaciliScope.

## Overview

Time Entries track labor time spent on work orders, calculate labor costs, and support productivity analysis.

## What are Time Entries?

Time Entries record the time spent by users on work orders. They track start and end times, duration, and calculate labor costs.

## Key Features

- **Time Tracking**: Record time spent on work
- **Labor Cost Calculation**: Automatically calculate labor costs
- **Work Order Integration**: Link to work orders
- **Productivity Analysis**: Analyze time spent on different tasks
- **Reporting**: Generate time reports

## Getting Started

1. Navigate to **Time Entries** in the main menu
2. Click **Create Time Entry** to log time
3. Select the work order
4. Enter start and end times
5. Add notes if needed
