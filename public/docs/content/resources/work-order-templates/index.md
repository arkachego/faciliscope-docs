# Work Order Templates

Complete guide to Work Order Templates in FaciliScope.

## Overview

Work Order Templates are reusable work order definitions that standardize common tasks, reduce data entry, and ensure consistency.

## What are Work Order Templates?

Work Order Templates are pre-configured work order definitions that can be used to quickly create new work orders with standard information.

## Key Features

- **Reusable Definitions**: Create once, use many times
- **Standardization**: Ensure consistent work order creation
- **Time Savings**: Reduce data entry time
- **Checklists**: Include standard checklists
- **Quick Creation**: Create work orders from templates quickly

## Getting Started

1. Navigate to **Work Order Templates** in the main menu
2. Click **Create Template** to add a new template
3. Define the template details
4. Add checklist items if needed
5. Save the template
