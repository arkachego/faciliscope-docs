# Budgets

Complete guide to Budgets in FaciliScope.

## Overview

Budgets are financial plans and allocations that track spending versus budget, support multiple budget types, and enable cost analysis.

## What are Budgets?

Budgets represent financial plans for facility management activities. They help track spending and ensure costs stay within allocated amounts.

## Key Features

- **Multiple Budget Types**: Support various budget categories
- **Spending Tracking**: Track actual spending vs. budget
- **Cost Analysis**: Analyze costs by category, facility, or time period
- **Budget Periods**: Define budget periods (monthly, quarterly, yearly)
- **Alerts**: Get notified when approaching budget limits

## Getting Started

1. Navigate to **Budgets** in the main menu
2. Click **Create Budget** to add a new budget
3. Select the budget type and period
4. Set the budget amount
5. Link to facilities or categories as needed
