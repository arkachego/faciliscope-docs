# Maintenance Schedules

Complete guide to Maintenance Schedules in FaciliScope.

## Overview

Maintenance Schedules are recurring maintenance plans that automatically generate work orders. They help ensure preventive maintenance is performed on time.

## What are Maintenance Schedules?

Maintenance Schedules define when and how often maintenance should be performed on assets. They automatically create work orders based on time intervals or usage.

## Key Features

- **Automatic Work Order Generation**: Create work orders automatically
- **Flexible Scheduling**: Schedule by time, usage, or calendar
- **Checklists**: Define standard procedures for each schedule
- **History Tracking**: Track all maintenance performed
- **Compliance**: Ensure maintenance meets requirements

## Getting Started

1. Navigate to **Maintenance Schedules** in the main menu
2. Click **Create Schedule** to add a new schedule
3. Select the asset to maintain
4. Define the schedule frequency
5. Add checklist items if needed
