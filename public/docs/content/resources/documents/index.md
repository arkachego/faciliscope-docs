# Documents

Complete guide to Documents in FaciliScope.

## Overview

Documents are files and attachments that can be organized by category, linked to resources, and version controlled.

## What are Documents?

Documents represent files, attachments, and documents related to facility management. They can be manuals, contracts, photos, or any other files.

## Key Features

- **File Storage**: Store and organize files
- **Category Organization**: Organize documents by category
- **Resource Linking**: Link documents to resources
- **Version Control**: Track document versions
- **Search**: Find documents quickly

## Getting Started

1. Navigate to **Documents** in the main menu
2. Click **Upload Document** to add a new document
3. Select the document category
4. Link to related resources if needed
5. Add tags for easy searching
